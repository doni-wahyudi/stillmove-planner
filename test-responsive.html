Responsive Design Testing

Instructions:

  • Resize your browser window to test responsive breakpoints
  • Test on mobile (< 768px), tablet (768px - 1024px), and desktop (> 1024px)

  • Check that navigation collapses to hamburger menu on mobile

  • Test with screen reader if available
